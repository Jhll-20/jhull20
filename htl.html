HTML:

HTML stands for HyperText Markup Language. It was created in 1991 by Tim Berners Lee at the CERN research institute in Switzerland. HTML is the standard markup language for web page development. HTML are essentially the “building blocks” of websites that define the composition of a website, such as sections, paragraphs, images and links. Just by itself HTML is capable of creating a static page, but when combined with CSS and JavaScript creates dynamic pages as seen commonly on the internet.


There are 5 versions of HTML listed down below with a description

Version Features and Info
HTML 1 Introduced in 1991. Was the first version of html and was only capable of basic elements for text documents.
HTML 2 Introduced in 1995. Introduced tables and early forms of user input and interactions.
HTML 3.2 Introduced in 1996. Added file upload support, early stage javascript with Applets and text flow enhancements
HTML 4 Introduced in 1998. Added Document Type Recognition, frames, scripting, and stylesheets
HTML 5 Introduced in 2014, 5.1 in 2016 and 5.2 in 2017. Added document structure elements such as "section" "article" "header" etc, SVG and MathMl markup support, audio and video support elements and new HTML APIs
